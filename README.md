Helper Application for the Board Game Nemesis Retaliation for Android Devices.

Working:

Hungarian, German, English, Polish language can be selected
Helps with the steps at each phase
Easier to teach the game to beginners
Can read out loud elements (togglable silent mode/phase names and rounds only/reading everything
Tracking Intruder Bag (can be toggled to retain physical bag handling). Can save time at the table and help avoid mistakes.
Can track rounds
Network mode with a Master and several clients
Port number can be edited to be enable multiple game sessions on the same Wifi Network.
There is a long press option on the intruder bag to manually add a token into it.
NemesisMainscreen
Segítő alkalmazás a Nemesis Retaliation társasjátékhoz Android eszközökre.

Működés:

Magyar, német, angol, lengyel nyelv választható
Segít az egyes fázisok lépéseiben
Könnyebbé teszi a játék tanítását kezdőknek
Hangosan felolvassa az elemeket (néma mód/csak fázisnevek és körök/minden felolvasása bekapcsolható)
Betolakodózsák követése (kikapcsolható a zsák fizikai kezelésének megőrzéséhez). Időt takaríthat meg az asztalnál és segít elkerülni a hibákat.
Követi a köröket
​​Hálózati mód egy Masterrel és több klienssel
A portszám szerkeszthető, hogy több játékmenet is lehetővé váljon ugyanazon a Wifi hálózaton.
A behatolózsákon van egy hosszú megnyomás opció, amellyel manuálisan hozzáadhatsz egy tokent.
